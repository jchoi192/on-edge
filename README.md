# OnEdge

An online learning tool that provides a platform for teachers and students to communicate effectively with each other.

## Getting Started
[Click Here](https://onedgeweb.herokuapp.com/) to get started on Heroku

## Technologies
- Python
- Django
- Postgresql
- HTML5
- CSS3
- JavaScript
- Materialize CSS
